# Ragul Portfolio

Personal portfolio for Ragul, featuring skills, GitHub projects, contact links, and automatically refreshed LeetCode progress.

Live site: [ragul-portfolio-iota.vercel.app](https://ragul-portfolio-iota.vercel.app/)

## Development

```bash
npm install
npm run dev
```

Use `npm run build:vercel` to create the production build used by Vercel.
